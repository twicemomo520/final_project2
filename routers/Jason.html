我的個人頁面
我的個人資料:
安裝tensorflow gpu流程

1.conda create -name env_name python=3.7

2.activate env_name

3.pip install --ignore-installed --upgrade env_name

刪除env: conda env remove --name env_name

pytorch安裝方法:

1.[使用pytorch網址安裝]

pytorch網址: https://download.pytorch.org/whl/torch_stable.html

ex. pip install https://download.pytorch.org/whl/cu101/torch-1.4.0-cp38-cp38-linux_x86_64.whl

2. [For pip]

install with specified version and -f (自己指定torch、torchvision版本)

ex. pip install torch==1.6.0 torchvision==0.7.0 -f https://download.pytorch.org/whl/cu101/torch_stable.html

pip install torch===1.4.0 torchvision===0.5.0 -f https://download.pytorch.org/whl/torch_stable.html

3.[For conda ]

Run conda install with cudatoolkit

依照cuda版本安裝匹配的torch、torchvision版本

ex. conda install pytorch torchvision cudatoolkit=10.1 -c pytorch

測試torch安裝成功否

import torch

torch.cuda.is_available()

安裝環境到jupyter

python -m ipykernel install --user --name env_name --display-name "Python (tensorflow-gpu)"

COCO API下載 https://github.com/philferriere/cocoapi

Tensorflow版本

最好是tensorflow-gpu=1.14.0 + keras=2.1.0

Pillow問題

不可直接使用pip install Pillow

應該是

1.pip uninstall PIL

2.pip uninstall Pillow

3.pip install Pillow

CUDA與cudnn版本

CUDA:10.1.243

cudnn:10.1 v7.6.5.32

Paris streetview dataset https://www.amuzinc.com/Product?kidCode=20191203002

scipy版本請安裝1.1.0， 1.3.0已經把imread從scipy.misc移除

pip install scipy==1.1.0

要額外安裝的包:

1.pip install scikit-image

2.pip install pandas

3.pip install tqdm
